Thattachu

Welcome to Thattachu. A simple typing tutor where you can learn typing in multiple keyboard layouts of multiple languages.

Step 1

Choose Language
Thattachu provides ability to learn typing for 62 languages supported by jQuery IME. Choose the language you want to learn.

Step 2

Choose Layout
Usually there are multiple keyboard layouts and input methods for a language. Choose the layout you want to practice.

Step 3

Choose Course
If there are available courses for the language and the input type you selected, we list them. You choose, you learn at your pace.
